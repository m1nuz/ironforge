# ironforge
Game engine

## Features ##
* OpenGL 3.3 support
* Forward rendering
* Phong materials
* Effects:
  * Normal mapping
  * Glow
  * Environment Mapping
* Texture streaming
* UI(True type fonts, widgets)
* Scene managment(json scene file)
* Lua scripting
* Support game controller input
